mai_ | Whisper to ChatGPT and Claude.ai

Die Erweiterung ermöglicht Sprachinteraktion mit ChatGPT und Claude.ai im Chrome-Browser und anderen auf Chromium basierenden Browsern (z.B. Edge). Sie ermöglicht die Verwendung des von OpenAI entwickelten AI-Modells "Whisper" für Sprache-zu-Text-Transkription. Sie können auch die in den Browser integrierte Transkriptionsmethode (webkitSpeechRecognition) verwenden. Die Erweiterung wird angezeigt und funktioniert beim Besuch von chat.openai.com oder claude.ai


Scrollen Sie nach unten, um mehr zu erfahren...

Programmfunktionen

  1. Sie können mit dem Chat sprechen, indem Sie ins Mikrofon sprechen, und seine Antworten werden laut vorgelesen

  2. Sie können eine Option aktivieren, die Ihr gesamtes Chat-Gespräch laut vorliest, oder nur die letzte Antwort laut vorliest

  3. Sie können einen Textabschnitt im Chat-Thread markieren und eine Funktion aktivieren, die diesen Abschnitt laut vorliest

  4. In der Erweiterungskonfiguration können Sie Sprachparameter festlegen, einschließlich der Sprache, in der Sie mit dem Chat sprechen, Stimme, Stimmhöhe und Lesegeschwindigkeit, Sprachtranskriptionsmethode

Benutzeroberfläche

Klicken Sie und diktieren Sie Ihre Frage mit Hilfe der Sprache-zu-Text-Technologie. Das Symbol ändert die Farbe zu Rot während der Sprachaufnahme. Klicken Sie erneut, um die Sprachaufnahme zu beenden und die Sprache in Text zu konvertieren. Wählen Sie Ihre bevorzugte Transkriptionsmethode in den Erweiterungseinstellungen aus. Details zu den verfügbaren Methoden finden Sie unten.
Mikrofon ist ausgeschaltet - Funktion "Sprache-zu-Text" ist nicht verfügbar.
Wenn diese Option aktiviert ist, wird die erscheinende Chat-Antwort automatisch vorgelesen. Sie können diese Option deaktivieren, indem Sie auf diese Schaltfläche klicken. Wenn das Vorlesen der Antwort gerade stattfindet (die Schaltfläche pulsiert rot), können Sie das Vorlesen unterbrechen, indem Sie auf diese Schaltfläche klicken.
Automatisches Vorlesen von Antworten ist deaktiviert. Klicken Sie, wenn Sie die Funktion erneut aktivieren möchten.
Klicken Sie einmal (einfacher Klick), um den gesamten Thread vom ersten Beitrag an vorzulesen. Klicken Sie schnell zweimal (Doppelklick), um den letzten Beitrag (Antwort) im Thread vorzulesen. Wenn die Thread-Lektüre gerade stattfindet - die Schaltfläche pulsiert rot - können Sie diese jederzeit unterbrechen, indem Sie auf die Schaltfläche klicken.
Sie können auch einen beliebigen Textabschnitt markieren und auf dieses Symbol klicken, um ihn laut vorzulesen.
Klicken Sie, um das Lesen des Threads (oder der Chat-Antwort) anzuhalten oder fortzusetzen.
Klicken Sie, um die Einstellungen der Erweiterung zu öffnen und anzupassen. Sie können die Einstellungen für die Sprach-zu-Text-Transkription, Sprachparameter für den TTS (Text-zu-Sprache) Synthesizer und andere Vorlieben konfigurieren. Weitere Informationen zur TTS-Konfiguration.

STT Sprachtranskription (speech-to-text)

Die Erweiterung bietet zwei Methoden zur Sprachtranskription:

  1. webkitSpeechRecognition - die Standardmethode, die die API des Chrome-Browsers verwendet. Die Transkription wird lokal (offline) durchgeführt. Hinweis: Diese Methode fügt keine Interpunktion hinzu und wird nur im Chrome-Browser unterstützt.

  2. Whisper - ein von OpenAI entwickeltes KI-Modell zur Sprache-zu-Text-Transkription. Es bietet eine hochwertige Transkription mit korrekter Interpunktion. Erfordert einen OpenAI-API-Schlüssel, was mit Kosten verbunden ist (OpenAI-Preisgestaltung). Die Transkription wird auf OpenAI-Servern durchgeführt, was das Senden der Tonaufnahme erfordert.

TTS-Sprachsynthese (text-to-speech)

  1. Die Erweiterung nutzt speechSynthesis - eine API, die vom Chrome-Browser und anderen auf Chromium basierenden Browsern (z.B. Edge, Opera, Brave) bereitgestellt wird. Dies ermöglicht die Sprachsynthese (TTS) im Offline-Modus, ohne Datenübertragung.
  2. In den Einstellungen können Sie die Sprache und Stimme für die Sprachsynthese auswählen sowie andere Parameter anpassen, einschließlich Stimmlage und Lesegeschwindigkeit.

Datenschutz und Benutzerdaten

Die Erweiterung sammelt oder übermittelt keine persönlichen Daten. Alle Einstellungen werden lokal in Ihrem Browser (localStorage) gespeichert. Wenn Sie jedoch die Transkriptionsmethode "Whisper" verwenden und Ihren OpenAI API-Schlüssel angeben, wird die Erweiterung mit den OpenAI-Servern kommunizieren, Audioaufnahmen zur Transkription senden und die OpenAI-Server werden den Transkriptionstext zurücksenden.
